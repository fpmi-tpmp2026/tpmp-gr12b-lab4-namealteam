# Глоссарий

- `Vertabelo` — сервис для проектирования ER-диаграмм базы данных.
- `SQLite` — встраиваемая SQL-база данных, используемая в проекте.
- `CRUD` — операции создания, чтения, обновления и удаления.
- `BLOB` — бинарный объект, используемый для хранения изображений автобусов.
- `Foreign Key` — внешний ключ, обеспечивающий ссылочную целостность.
- `GitHub Pages` — статический хостинг для публикации сайта проекта.
- `Wiki` — набор связанных Markdown-страниц с документацией.
